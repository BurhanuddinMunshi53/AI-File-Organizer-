# ⬡ AETHER OS — AI Files Organizer

> "SENTINEL UI. JARVIS CORE. ULTRA-PURE SECURITY."

A terminal-based intelligent file management system wrapped in a dark cyberpunk HUD. 12 powerful operations, 120+ file type recognition, built for Android Termux and desktop.

## Features
- Neural Category Sort — auto-classifies files into 12 smart categories
- Secure Node Vaporizer — targeted file deletion
- Deep Search Frequency — recursive keyword search
- Encrypt / Decrypt — obfuscate files with .stark extension
- Mass Identity Rewrite — batch rename with custom prefix
- Metadata Reconnaissance — file size per directory
- Data Integrity Check — MD5 fingerprint per file
- Sector Clone Redundancy — full folder backup
- Sector Flatten & Purge — collapse nested dirs, purge empty shells
- System Volume Diagnostics — live disk usage

## Installation
pip install nothing — pure Python stdlib
python AI_Files_Organizer.py

## Requirements
Python 3.7+ · Android Termux or any ANSI terminal
