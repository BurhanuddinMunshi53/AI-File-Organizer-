# ⬡ AETHER OS — AI Files Organizer

> *"SENTINEL UI. JARVIS CORE. ULTRA-PURE SECURITY."*

A terminal-based intelligent file management system wrapped in a dark cyberpunk HUD. 12 powerful operations, 120+ file type recognition, built for Android Termux and desktop environments.

---

## ⚡ Commands

| Code | Command | Description |
|------|---------|-------------|
| `01` | Scan Entire System | Maps all root directories and main folders |
| `02` | Neural Category Sort | Auto-classifies files into 12 smart categories |
| `03` | Secure Node Vaporizer | Targeted single-file deletion |
| `04` | Deep Search Frequency | Recursive keyword search across entire filesystem |
| `05` | Encrypt Data Sector | Obfuscates all files with .stark extension |
| `06` | Decrypt Data Sector | Strips .stark and restores original files |
| `07` | Mass Identity Rewrite | Batch rename with custom prefix and index |
| `08` | Metadata Reconnaissance | File size intelligence per directory |
| `09` | Data Integrity Check | MD5 fingerprint display per file |
| `10` | Sector Clone Redundancy | Full folder deep-copy backup |
| `11` | Sector Flatten & Purge | Collapse nested directories, purge empty shells |
| `12` | System Volume Diagnostics | Live total, used, and free disk stats |
| `XX` | Terminate | Graceful JARVIS-style shutdown sequence |
